Gomba
=====

Simple platformer in Processing & Processing.js

Building and Running
--------------------
```shell
$ git clone git://github.com/asalga/Gomba.git
$ cd Gomba
$ make
$ python -m SimpleHTTPServer
```

... and then open a browser and go to `http://localhost:8000`

Playing
